# 67. Add Binary

## Description
Given two binary strings `a` and `b`, return *their sum as a binary string*.

#### Example 1:
```
Input: a = "11", b = "1"
Output: "100"
```
#### Example 2:
```
Input: a = "1010", b = "1011"
Output: "10101"
```


## Solution

### Python
```python
class Solution:
    def addBinary(self, a: str, b: str) -> str:
        def binary_n(num):
            ans = ""
            while num > 0:
                ans = str(num % 2) + ans
                num //= 2
            return ans if ans else "0"
        
        return binary_n(int(a, 2)+int(b, 2))
```
